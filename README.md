# GoldenZetsu
A claude Code marketplace for fun plugins

## Plugins

### Claude Phrase Catcher
Catches Claude's commonly used phrases and shows an image and plays audio when they come in the conversation.

#### Installation
/plugin marketplace add PrinceBobo/GoldenZetsu

/plugin install Claude-Phrase-Catcher@goldenZetsu-marketplace

#### Enable auto-updates (recommended)
Go to /plugin → find goldenzetsus-marketplace → Enable auto-update
